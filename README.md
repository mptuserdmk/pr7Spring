# Практическая работа №5 (Spring Boot)

Данный проект представляет собой CRM-систему для барбершопа (BarberShop CRM), разработанную в рамках **практической работы №5**.

## О проекте
Приложение реализовано на стек:
- **Core**: Java 17+, Spring Boot 4
- **ORM / Database**: Spring Data JPA, Hibernate, PostgreSQL
- **UI / Templates**: Thymeleaf, HTML5, CSS3

## Функционал
- **CRUD операции** для основных сущностей:
  - Клиенты/Пользователи (`User`)
  - Услуги (`Service`)
  - Портфолио специалистов (`Portfolio`)
  - Категории услуг (`Category`)
  - Записи на стрижку (`Appointment`)
- Настроенные **связи между сущностями** в БД (One-to-Many / Many-to-One).
- Выбор внешних ключей в формах по понятным логическим атрибутам (название категории, email пользователя, название услуги) вместо безликих ID.
- Корректный вывод связанных сущностей в таблицах и списках.

## Запуск
1. Настройте подключение к СУБД PostgreSQL в `src/main/resources/application.properties`.
2. Запустите проект с помощью Maven:
   ```bash
   ./mvnw spring-boot:run
   ```
3. Откройте в браузере: `http://localhost:8080`
